## Audio Requirements
- Duration: 1-3 seconds max (notification-style short sounds)
- Format: WAV or MP3
- Clean, single note/hit sounds
- Naming: `{instrument}_{note}.wav` (e.g., `piano_C.wav`, `drums_kick.wav`)

## File Naming Convention
- Piano: `piano_C.wav`, `piano_D.wav`, ..., `piano_B.wav`
- Guitar: `guitar_C.wav`, `guitar_D.wav`, ..., `guitar_B.wav`
- Drums: `drums_kick.wav`, `drums_snare.wav`, `drums_hihat.wav`, `drums_tom.wav`, `drums_crash.wav`, `drums_ride.wav`
- Violin: `violin_G.wav`, `violin_A.wav`, ..., `violin_F.wav`
- Flute: `flute_C.wav`, `flute_D.wav`, ..., `flute_B.wav`
- Trumpet: `trumpet_C.wav`, `trumpet_D.wav`, ..., `trumpet_B.wav`
