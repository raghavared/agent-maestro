# Command Palette & Keyboard Shortcuts

**Navigate, create, and control everything without leaving the keyboard.**

---

## Command Palette

Press `Cmd+K` (macOS) or `Ctrl+K` (Windows/Linux) to open the command palette. It's the fastest way to reach anything in the app.

[screenshot placeholder: Command palette search bar with filtered results showing tasks, sessions, and actions]

The command palette lets you:
- Search for tasks by name
- Jump to sessions
- Execute actions (new task, new session, toggle panels)
- Access settings and configuration

Start typing to filter results. Use arrow keys to navigate and Enter to select.

## Agent Shortcuts

Agent shortcuts are pre-configured workflows you can launch with a single click or keystroke. Access them from the sidebar's session controls by clicking **Agent Shortcuts**.

[screenshot placeholder: Agent Shortcuts modal showing a list of configured shortcut workflows]

Each shortcut defines:
- A name and description
- The command or process to execute
- Optional parameters (model, skills, task assignment)

Shortcuts are customizable — create your own for workflows you repeat often.

---

## Quick Prompts

Quick prompts are one-click messages you can fire at the active terminal session. They appear in the sidebar's prompts section.

[screenshot placeholder: Quick Prompts section in sidebar showing pinned prompts with send buttons]

- **Send a prompt** — Click it, or use `Cmd+1` through `Cmd+5` for the first five
- **Edit a prompt** — Click the edit icon to modify the prompt text
- **Manage prompts** — Open the full prompts panel with `Cmd+Shift+P`

Useful for repetitive instructions like:
- "Run the test suite"
- "Fix all linting errors"
- "Summarize what you've done"
- "Commit your changes"

## Tips

- The command palette is context-aware — it shows results relevant to the current project
- Use keyboard shortcuts to stay in flow — switching sessions with `Cmd+E` / `Cmd+R` is much faster than clicking
- Pin your 5 most-used prompts so they're always one keystroke away
- The `Esc` key dismisses any open modal, panel, or overlay
